A decentralized crowdfunding application designed to empower women entrepreneurs. The platform leverages blockchain technology to ensure transparency in contributions and automated fund distribution.

### Technical Stack
* **Blockchain:** [Tezos](https://tezos.com/), SmartPy
* **Frontend:** React.js, JavaScript
* **Tools:** Temple Wallet, Taquito

### Core Capabilities
* Smart Contract Engineering: Developed secure smart contracts using **SmartPy** to automate contribution tracking, goal validation, and secure fund transfers on the Tezos blockchain.
* Dynamic Frontend: Built responsive UI components in **React** with integrated wallet connection and real-time transaction status updates.
* Payment Orchestration: Integrated seamless payment workflows allowing users to contribute XTZ (Tezos) directly to verified business campaigns.
